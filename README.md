🍔 Hungry App - Burger Delivery Application
A modern food delivery mobile application built with Flutter that allows users to browse, customize, and order their favorite burgers. The app communicates with a Backend API to fetch real-time product data and manage user authentication.

✨ Key Features
Dynamic Product Listing: Fetches the latest burger menu and categories directly from an API.

Customization (Toppings): Users can customize their sandwiches with extra toppings and sauces.

Shopping Cart Management: Add, remove, and update item quantities with real-time price calculation.

Secure Authentication: Complete Login and Registration system for user accounts.

Checkout & Payment: Integrated checkout flow including payment method selection (Cash/Card) and address management.

User Profile: Dedicated profile view to manage personal information and view order history.

Splash Screen: Branded animated entry to enhance the user experience.

🚀 Tech Stack
Frontend: Flutter & Dart.

Networking: Dio (for API integration).

Local Storage: Shared Preferences (for saving user tokens).
